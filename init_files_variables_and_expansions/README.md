This project contains a collection of Bash scripts created to practice shell fundamentals such as text processing, environment variables, arithmetic operations, file manipulation, and command-line utilities.

Each script is designed to perform a specific task using standard Unix tools and Bash built-ins, with restrictions on certain commands to reinforce low-level shell understanding. 
===================================================================================

0-alias
Purpose

Creates an alias that replaces the ls command with a destructive command.

1-hello_you
Purpose

Prints a personalized greeting using the current system user. 

2-path
Purpose

Modifies the PATH so /action is searched last when executing commands. 

3-paths
Purpose

Counts how many directories exist in the PATH variable. 

5-local_variables
Purpose

Prints all environment variables. 

4-global_variables
Purpose

Shows shell variables, environment variables, and functions. 

6-create_local_variable
Purpose

Defines a local variable inside the shell session. 

7-global_variable
Purpose

Creates an environment variable accessible to child processes. 

8-ture_knowledge
Purpose

Adds 128 to an environment variable. 

9-divide_and_rule
Purpose

Divides two environment variables. 

10-love_exponent_breath 
Purpose

Raises a number to a power. 

11-binary_to_decimal
Purpose

Converts binary number to decimal. 

15-rot13
Purpose

Encodes or decodes text using ROT13 cipher. 

14-decimal_to_hexadecimal
Purpose

Displays alternating lines from input.

13-print_float
Purpose

Formats a number with 2 decimal places. 

14-decimal_to_hexadecimal
Purpose

Converts base-10 number to base-16.
